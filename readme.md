# Simple flight manager
Simple REST API app for managing flights. You can create a flight, set its destination and price and then assign clients to it!

**Technology**
- Java 8
- Spring Boot 2.1.6
- MySQL
- JUnit
- Swagger2
- Angular 8

Deployed on Heroku.
